# Timeline - Microsoft Power Platform

## Visão Geral

Este projeto apresenta a evolução da Microsoft Power Platform ao longo dos anos por meio de uma timeline interativa. 
Ele destaca os principais marcos da plataforma, desde o lançamento do PowerApps e Power Automate (antigo Flow) até a introdução de inteligência artificial com o Copilot.

A página foi desenvolvida com um design responsivo e interativo, utilizando HTML, CSS e JavaScript, além da biblioteca Bootstrap para otimizar a estilização e o layout.

## Tecnologias Utilizadas

O projeto foi desenvolvido com as seguintes tecnologias:

- **HTML5** - Estruturação da página
- **CSS3** - Estilização e layout responsivo
- **Bootstrap 5.3** - Componentes visuais e estilização
- **JavaScript (Vanilla JS)** - Animações e interatividade da timeline
- **Google Fonts** - Utilização da fonte "Segoe UI"

## Estrutura de Arquivos

```
projeto-timeline-powerplatform
│-- index.html           # Página principal do projeto
│-- style.css            # Estilos principais da aplicação
│-- script.js            # Código JavaScript para animações
│-- images/              # Pasta com imagens utilizadas na timeline
│-- README.txt           # Documento explicativo do projeto
```

## Explicação dos Arquivos

- **index.html** - Contém toda a estrutura da página, incluindo a timeline, sumário e referências.
- **style.css** - Define os estilos visuais da timeline, sumário e referências, garantindo um layout moderno e responsivo.
- **script.js** - Adiciona interatividade, como animações ao rolar a página e efeitos visuais nos eventos da timeline.

## Como Executar o Projeto

1. Faça o download ou clone o repositório:
   ```sh
   git clone https://github.com/seu-usuario/projeto-timeline-powerplatform.git
   ```

2. Abra o arquivo **index.html** em qualquer navegador moderno.

## Funcionalidades do Projeto

- **Timeline interativa** - Exibição de eventos históricos com animações suaves.
- **Sumário dinâmico** - Navegação rápida entre os eventos da timeline.
- **Seção de referências profissional** - Organização dos links com destaque visual.
- **Design responsivo** - Adaptável para diferentes dispositivos e tamanhos de tela.

## Objetivo do Projeto

O projeto foi criado para ilustrar a evolução da Microsoft Power Platform, destacando seus principais marcos e avanços tecnológicos.
Ele pode ser utilizado como material educacional, referência acadêmica ou portfólio de desenvolvimento web.
