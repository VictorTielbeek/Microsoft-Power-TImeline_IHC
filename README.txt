📌 Timeline - Microsoft Power Platform

📖 Visão Geral
Este projeto tem como objetivo apresentar a evolução da Microsoft Power Platform ao longo dos anos por meio de uma timeline interativa.
Ele destaca os principais marcos, desde o lançamento do PowerApps e Power Automate (antigo Flow) até a incorporação de inteligência artificial com o Copilot.

A página foi desenvolvida com um design moderno e responsivo, utilizando HTML, CSS e JavaScript, além da biblioteca Bootstrap para facilitar a estilização e layout.

🚀 Tecnologias Utilizadas
O projeto foi desenvolvido utilizando as seguintes tecnologias:

- HTML5 → Estruturação da página
- CSS3 → Estilização e layout responsivo
- Bootstrap 5.3 → Componentes visuais e estilização
- JavaScript (Vanilla JS) → Animações e interatividade da timeline
- Google Fonts → Fonte personalizada "Segoe UI"

📂 Estrutura de Arquivos

📂 projeto-timeline-powerplatform
│-- 📄 index.html           # Página principal do projeto
│-- 📄 style.css            # Estilos principais da aplicação
│-- 📄 script.js            # Código JavaScript para animações
│-- 📂 images               # Pasta com imagens utilizadas na timeline
│-- 📄 README.txt           # Documento explicativo do projeto

📌 Explicação dos Arquivos

- index.html → Contém toda a estrutura da página, incluindo a timeline, sumário e referências.
- style.css → Define os estilos visuais da timeline, sumário e referências, garantindo um layout moderno e responsivo.
- script.js → Adiciona interatividade, como animações ao rolar a página e efeitos visuais nos eventos da timeline.

💡 Como Rodar o Projeto

1. Baixe ou clone o repositório
   git clone https://github.com/seu-usuario/projeto-timeline-powerplatform.git

2. Abra o arquivo index.html no navegador.

📜 Funcionalidades do Projeto

✅ Timeline Interativa → Animações suaves ao rolar a página.
✅ Sumário Dinâmico → Links para cada evento da timeline.
✅ Seção de Referências Profissional → Links organizados com destaque visual.
✅ Layout Responsivo → Adaptável para diferentes dispositivos.
